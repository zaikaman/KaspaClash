# Cosmetic Asset Prompts & Item Definitions

This document contains all cosmetic items for the KaspaClash shop system, with image generation prompts for each asset.

---

## General Guidelines

- **Style**: Cyberpunk fighting game aesthetic with Kaspa teal (#49EACB) accents
- **Format**: WebP, 512x512px for thumbnails, 1024x1024px for previews
- **Background**: Transparent preferred, dark gradient fallback
- **Base Classes**: Characters are mapped to base archetypes (Ninja, Bruiser, Warrior, Hunter) for gameplay animations.

---

```
A crystalline shard currency icon for a cyberpunk fighting game. The shard is a geometric hexagonal crystal with faceted surfaces, glowing with electric teal (#49EACB) and gold (#F0B71F) energy.
```
